Removes components from entities via world.remove_componentsBRP method. Useful for removing behaviors, implementing state machines, and optimizing entities.

Example:
```json
{
  "entity": 123,
  "components": [
    "bevy_sprite::sprite::Sprite",
    "bevy_render::camera::camera::Camera"
  ]
}
```

Notes:
- Removing non-existent components isn't an error
- Entity persists even if all components removed
- Use world_despawn_entity to remove entity entirely
