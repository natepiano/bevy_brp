Inserts/replaces components on entities via `world.insert_components` BRP method. Useful for adding behaviors, modifying capabilities, and implementing state changes.

Behavior: Replaces existing components, adds new ones, preserves others.

Example:
```json
{
  "entity": 123,
  "components": {
    "bevy_sprite::sprite::Sprite": {
      "color": {"r": 1.0, "g": 0.0, "b": 0.0, "a": 1.0}
    }
  }
}
```

Note: Requires component to be registered with BRP and have the Reflect trait
