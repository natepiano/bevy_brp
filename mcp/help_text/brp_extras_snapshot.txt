Recursive YAML outline of a Bevy UI entity tree via bevy_brp_extras.

Behavior:
- Walks the UI tree from `root` (or every top-level `Node` if omitted)
- Each row: full entity id (Entity::to_bits, u64), a short component-type label, any text content, and a `name: "#Foo"` line (BSN entity-reference syntax) when the entity has a `Name` component

Prerequisites:
- bevy_brp_extras dependency required
- BrpExtrasPlugin must be registered
