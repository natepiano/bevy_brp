Exhaustively list BRP transport method names registered on a running Bevy app.

Bevy 0.19's OpenRPC entries include built-in and application-defined methods, but provide no useful
parameter/result schemas, descriptions, or examples. For curated developer guidance and raw
schemas, use brp_list_agent_tools, then pass the selected exact method and raw params to
brp_execute. Most discovered methods need not be in that curated subset.

The response also contains BRP server information when the HTTP transport provides it.
